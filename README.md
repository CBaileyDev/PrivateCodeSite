# PrivateCodeSite

Marketing website for [PrivateCode](https://github.com/CBaileyDev/PrivateCode) — the native, instant, local-first AI coding agent.

## Live Site

Open `index.html` in a browser (or enable GitHub Pages).

## Features
- Real official screenshots from the PrivateCode repo
- Scroll-triggered animations
- Interactive code intelligence demo
- Terminal safety demo
- Comparison table
- Beautiful dark theme matching the app

Built with Tailwind + custom CSS, AOS-style reveals, and real product information.
